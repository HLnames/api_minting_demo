# Hyperliquid Names Minting Demo

A demonstration application for minting .hl names using the HLnames API. Supports both testnet and mainnet.

Reach out to us if you have any questions or want to integrate!

## Getting Started

Install dependencies and run the development server:

```bash
pnpm install
pnpm run dev
```
